# Lyaris Client

Minecraft auto-mining bot with web interface.

## Setup

```bash
npm install
```

Configure `.env`:
```
BOT_HOST=localhost
BOT_PORT=25565
BOT_USERNAME=LyarisBot
MC_VERSION=1.20.4
VIEWER_PORT=3007
```

## Run

```bash
npm start
```

- Control panel: http://localhost:3007
- 3D viewer: http://localhost:3008

## Usage

1. Start the bot — it connects to the Minecraft server
2. Open the control panel in your browser
3. Enter two corner coordinates to define the mining area
4. Click **Start** — the bot mines the area automatically
5. Use **Pause / Resume / Stop** to control the process
